# @r4pm/components

React components for process mining: Petri nets, object-centric Petri nets,
directly-follows graphs, dotted charts, alignments, etc.

## Install

```sh
pnpm add @r4pm/components
# peer dependencies:
pnpm add react react-dom @tanstack/react-query
```

## Use

Wrap your app (or any single viewer) in a Radix `Theme`, import the stylesheet once, then
feed a viewer the typed binding result.

```tsx
import "@r4pm/components/styles.css";
import { Theme, PetriNetViewer } from "@r4pm/components";

export function App({ net }) {
  return (
    <Theme>
      <PetriNetViewer data={net} />
    </Theme>
  );
}
```

Coloring, value formatting, and interactivity are shared across viewers and configured once.
See [Viewer Configuration](?path=/docs/getting-started-viewer-configuration--docs). For the exact
shape of each viewer's `data` prop (the binding return types, with field descriptions), see
[Data Types](?path=/docs/getting-started-data-types--docs).
