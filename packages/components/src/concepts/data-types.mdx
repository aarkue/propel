# Data types

Reference for the typed values viewers receive as their `data` prop. Scoped to just those types (the binding return types a viewer renders, plus what they reference) rather than every binding. They're generated from Rust via schemars (JSON Schema) into `@r4pm/client`; the descriptions are the original Rust doc comments. Type names in a field's type cross-reference other entries on this page.

See also [Introduction](?path=/docs/getting-started-introduction--docs) and [Viewer Configuration](?path=/docs/getting-started-viewer-configuration--docs).
